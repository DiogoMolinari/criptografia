# 🔐 Criptografia em Python
Projeto desenvolvido para aplicar conceitos de lógica de programação e criptografia básica utilizando Python.

## ⚙️ Funcionalidades
* Criptografia de mensagens
* Descriptografia utilizando a mesma chave
* Interface via terminal

## 🛠️ Tecnologias
* Python

## ▶️ Como executar
```bash
python main.py
```
